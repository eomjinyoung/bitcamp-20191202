# 팀 프로젝트 개요

교육과정 중 수행하는 미니 팀 프로젝트이다.

- 기간
  - 교육과정에 따라 5주에서 10주 사이의 기간 동안 프로젝트를 수행한다.
- 진행 계획
  - [프로젝트 안내](plan-guides.md)에 따라 진행한다.
